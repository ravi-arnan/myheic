# MyHeic

Konversi file HEIC ke JPG di komputer Windows. Gratis, offline, tanpa batas.

HP Samsung dan iPhone modern menyimpan foto dalam format HEIC supaya hemat memori, tapi banyak laptop/PC tidak bisa membukanya tanpa software tambahan. MyHeic adalah aplikasi desktop ringan untuk mengubah HEIC menjadi JPG — semuanya berjalan di komputer kamu, tanpa upload ke server.

## Fitur

- Drag & drop file HEIC/HEIF ke window
- Batch conversion (banyak file sekaligus)
- Quality slider 10–100
- Output ke folder asal atau folder pilihan
- Auto-rename kalau file output sudah ada
- 100% offline — foto tidak pernah keluar dari komputer

## Download

Cek tab [Releases](../../releases) untuk installer Windows terbaru.

## Pengembangan

```bash
pnpm install
pnpm dev          # jalankan di mode development
pnpm build:win    # build .exe (Windows runner)
```
